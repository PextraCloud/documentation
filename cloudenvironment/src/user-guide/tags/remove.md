# Remove a Tag

To remove a tag from a resource in Pextra CloudEnvironmentÂ®:

1. Locate the tag you want to remove.
2. Click the **X** icon on the tag.

![Remove Tag](./images/tag-remove-1.png)

> [!TIP]
> Removing a tag does not affect the resource itself, but it may impact any automation, filtering, or policies that rely on that tag.
