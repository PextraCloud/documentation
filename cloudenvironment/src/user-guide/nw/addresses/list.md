# List Node Addresses
View managed and staged addresses for a node.

> [!NOTE]
> Pending changes are visible here, but are not applied until you [Apply Changes](./apply.md).

## Web Interface
1. Click a node in the left navigation.
2. Select the **Network** tab.
3. Open **Addresses**.
