# Policy Management

This section provides an overview of **policy management** in Pextra Cloud EnvironmentÂ®. It describes how administrators can create and manage **custom user-defined policies**. Access to this module requires **administrative privileges**.

## Accessing Policies

1. In the **left panel**, select your **organization**.
2. In the **right panel**, click the **IAM** tab.
3. Navigate to the **Policies** tab.

![View Policy](./images/policy-create-1.png)

> [!TIP]
> Only users with administrative privileges can create, modify, or delete policies. Ensure you have the necessary permissions before attempting to manage policies.
