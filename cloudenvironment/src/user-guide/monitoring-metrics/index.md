# Monitoring & Metrics

Pextra CloudEnvironment® provides a comprehensive **Monitoring & Metrics** dashboard, allowing administrators to observe and manage the health, performance, and resource usage of clusters, nodes, and instances.

## Viewing Available Monitoring Metrics

To access monitoring metrics:

1. Select the **cluster**, **node**, or **instance** in the left navigation panel.
2. On the right-hand panel, the **Monitoring** section displays key metrics, organized under three main categories: **Cluster**, **Node**, and **Instance**.

> [!TIP]
> To view monitoring metrics, click the specific **resource** (cluster, node, or instance) in the left navigation panel. This opens the monitoring dashboard for that resource on the right-hand panel.
